# 🧠 Daily Intel

**Daily AI + Security Intelligence Feed by Manthan Ghasadiya**

Automated daily research digest covering AI security, red teaming, MCP vulnerabilities, general cybersecurity, and AI research — curated by an autonomous red team operator.

## Structure

```
intel/
├── README.md
├── daily/
│   └── YYYY/MM/DD.md          (daily digest files)
└── papers/
    └── YYYY/MM/
        └── paper-title-slug.md (research paper deep dives)
```

- **`daily/`** — Concise daily digests with headlines, technical breakdowns, and actionable takeaways
- **`papers/`** — Deep-dive breakdowns of individual research papers, linked from daily digests

## Coverage

- AI Security & Red Teaming
- MCP (Model Context Protocol) Security
- AI Agent Security & Vulnerabilities
- General Cybersecurity (breaches, CVEs, attack techniques)
- AI Research (models, frameworks, papers)
- Trending Security & AI Tools on GitHub
- Paper deep dives with methodology, results, and relevance analysis

## Purpose

Curated, technically deep intelligence for AI red teaming, MCP security scanning, CVE research, autonomous pentesting tool development, and staying ahead of the offensive AI curve.
