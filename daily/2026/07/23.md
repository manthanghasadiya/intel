### [📄 Terence Tao's ChatGPT Conversation About the Jacobian Conjecture Counterexample](https://chatgpt.com/share/6a5fdc7a-d6f8-83e8-bbea-8deb42cfed56)

### [📄 Cactus Hybrid: Gemma 4 On-Device Model Knows When It's Wrong](https://github.com/cactus-compute/cactus-hybrid)
**⏱ Read time:** 3 min

Cactus posts-trains Gemma 4 E2B with internal confidence probes (0-1 score per answer, returned as structured data). When confidence < 0.85, the query is routed to Gemini 3.1 Flash-Lite. The hybrid model matches Flash-Lite benchmarks while running only 15-35% of queries on the cloud — the rest runs on-device. Open source under MIT license, with MLX integration for Apple Silicon.

**Why it matters:** Confidence-based hybrid routing is a practical architectural pattern for agent security — enables graceful degradation and guardrails when models are uncertain, which is exactly when they're most vulnerable to adversarial manipulation.

## 🔌 MCP / AI AGENTS

### [📄 ANSI Escape Sequence Injection in MCP Servers: Hidden from Humans, Visible to AI](https://brightsec.com/research/detecting-ansi-escape-sequence-injection-in-mcp-servers-with-dast/)
**⏱ Read time:** 5 min

Bright Security researchers detail ANSI Escape Sequence Injection (AESI) attacks against MCP servers. Two variants: (1) Direct-fetch AESI — attacker hosts content with hidden ANSI codes at a URL, MCP fetch tool retrieves it, model reads concealed instructions; (2) Stored AESI — payload written into storage through one entrypoint, detonates when a different entrypoint reads it. Model-consumable fields in MCP (result.content[].text, result.contents[].text, result.messages[].content.text) are the injection surface. The article provides a DAST detection methodology using Bright scanner.

**Why it matters:** Directly relevant to Manny's red teaming of MCP servers. AESI is a practical prompt injection variant that bypasses human review. The stored variant (write-once, detonate-later) is especially dangerous for multi-user MCP deployments.
